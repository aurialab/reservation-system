Autentica l'usuari mitjançant credencials (Email/Password). Si les dades són vàlides, el servei genera i retorna un JSON Web Token (JWT) que el client haurà d'incloure en la capçalera Authorization per a les peticions protegides (com la reserva d'activitats). El sistema valida les credencials contra la base de dades xifrada.
